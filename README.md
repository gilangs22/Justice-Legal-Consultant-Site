# Justice Legal Consultant

Website company profile dan katalog layanan untuk Justice Legal Consultant (JLC).

## Jalankan lokal

```bash
npm install
npm run dev
```

Buka `http://127.0.0.1:3000`.

## Konten utama

- Profil singkat JLC
- Katalog layanan legalitas, properti, PBB, perizinan, badan usaha, dan dokumen hukum
- Dokumen awal perseorangan dan perusahaan
- Form konsultasi yang langsung membuka WhatsApp
- Area portofolio/testimoni untuk foto dokumen yang sudah disamarkan
